# AI Agents with Google ADK and Gemini

This repository contains example code for **Google’s 5-Day AI Agents Intensive (Nov 2025)** on **Kaggle**.  
The course teaches building autonomous AI agents using **Google’s Agent Development Kit (ADK)** and **Gemini LLMs**, covering topics such as:

- Agent architectures and tools  
- Multi-agent workflows  
- Memory and evaluation  
- Team orchestration of agents
